Tokenized Wallets

Hot Potatoes are NFTs with an integrated wallet that can only be controlled by its current owner. Turn your cross-chain portfolio into a tradable and programmable asset.

Secured with Threshold ECDSA

Warning: Hot Potatoes are still in early alpha and the bridge contract is still upgradable. Only use it for testing and development purposes and at your own risk.
  • All signatures are authorized on an EVM blockchain (currently Arbitrum). If an unauthorized transaction is made, there will be a cryptographically verifiable fraud proof.
  • There are no central private keys. Signatures are created using the Internet Computer's Threshold ECDSA technology. The Internet Computer is a network of small blockchains run by vetted companies (13 nodes for regular subnets, 34 nodes for the fiduciary subnet).
  • For a faster user experience, the current version trusts the Arbitrum sequencer to be honest instead of waiting for L1 finality (7 days) or even L1 confirmation.
  • The current version verifies EVM state via well-known RPC providers (e.g. Alchemy). Future versions might use cryptographic proof, e.g. by running a light client of the beacon chain on the Internet Computer.
